True Fantøm's Extension Gallery

Unlike custom extensions on other websites, these aren't limited by the extension sandbox, so they are a lot more powerful. All extensions are reviewed for safety.

To use multiple of these extensions in Scratch mods like TurboWarp, hover over the extension and press the button to download its source code. Then go to the editor, open the extension chooser, then choose the "Custom Extension" option at the bottom, then choose the "File", and select the downloaded code. Also check the box "Run extension without sandbox" to make everything work.

These extensions are not compatible with Scratch.
Projects that use these extensions can't be uploaded to the Scratch website. They can, however, be used in packagers like this.
